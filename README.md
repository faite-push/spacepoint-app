# SpacePoint App 🚀

O **SpacePoint App** é o frontend moderno da plataforma SpacePoint, construído com as tecnologias mais recentes para proporcionar uma experiência de usuário premium, rápida e intuitiva.

## 🛠️ Tecnologias Utilizadas

- **React 19 & Next.js 16 (Turbopack)**: Base da aplicação para performance e SEO.
- **Tailwind CSS 4**: Estilização moderna e otimizada.
- **TanStack React Query**: Gerenciamento de estado de dados e sincronização com a API.
- **Zustand**: Gerenciamento de estado global leve.
- **Framer Motion**: Animações fluidas e interativas.
- **Lucide React**: Biblioteca de ícones consistente.
- **React Hook Form & Zod**: Validação de formulários robusta.
- **ShaDCN & Radix UI**: Componentes de interface acessíveis e altamente customizáveis.
- **TinyMCE / Tiptap**: Editor de texto rico para o painel administrativo.

## 📦 Funcionalidades

- **Dashboard Administrativo Consolidado**: Gestão completa da loja em um único lugar.
- **Gestão de Banners e Reviews**: Personalização total da vitrine inicial.
- **Controle de Estoque e Produtos**: Gerenciamento detalhado de variantes e inventário.
- **Sistema de Cupons**: Criação e monitoramento de promoções.
- **SEO Otimizado**: Configurações técnicas acessíveis via dashboard.
- **Design Premium**: Interface escura (dark mode) com foco na estética e usabilidade.

## 🚀 Como Iniciar

1. Instale as dependências:
   ```bash
   pnpm install
   ```

2. Configure as variáveis de ambiente (.env.local):
   ```env
   NEXT_PUBLIC_API_URL=http://localhost:5000
   ```

3. Inicie o servidor de desenvolvimento:
   ```bash
   pnpm run dev
   ```

4. Acesse `http://localhost:3000`
